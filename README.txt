Github repos search engine

Search for github user and display his repos. Displaying 20 repos at a time, with ability to load addtional 20 repos per single request.

Installing notes

Known problem is no compatibility of Jest with babel-core. 
To fix the problem, inside of node_modules folder move babel-core inside jest configuration folder to get the correct version of babel core to cooperate with Jest and enable tests.
